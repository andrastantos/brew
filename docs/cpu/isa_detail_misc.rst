.. _rd_eq_fpstat:

$rD <- fpstat
--------------

*Instruction code*: 0x.008

*Exceptions*: none

*Type variants*: No

Description
~~~~~~~~~~~

:code:`$rD` is loaded by the current value of the :ref:`fpstat<fpstat>` register.

Description

The :ref:`fpstat<fpstat>` register is loaded from :code:`$rD`.
